Image Search API

This simple image search API utilises Google Custom Search Engine to return results for images related to a search phrase provided by the user, as shown in the first example API call below. The results returned are similar to performing a regular image search through Google, with the exception that strict safe search is applied.

In addition to the image search API, search phrases are stored in a database and the 10 most popular search phrases that have been passed through the API can be retrieved using the second example API call.

Example API Calls:

https://honmanyau-fcc-image-search-api.glitch.me/api?q=Nyancat&page=1 https://honmanyau-fcc-image-search-api.glitch.me/top
